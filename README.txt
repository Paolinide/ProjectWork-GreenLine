Come usare il programma:

1: avviare database di influx
2: avviare la console di influx
3: aprire progetti

    -testNodeServer
    -ProgettoAutobus

4: avviare prima testNodeServer
5: avviare ProgettoAutobus
6: avviare postman
7: select da console influx per verificare dati inseriti
8: fare GET all'indirizzo http://localhost:3000/get per verificare body
